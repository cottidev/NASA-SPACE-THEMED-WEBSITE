# NASA-SPACE-THEMED-WEBSITE

TECH STACK I USED:
HTML, CSS & JAVASCRIPT

CREATED WITH HELP FROM CLAUDE,

here are some feautures

1. 🖱️ Custom Cursor — The default cursor is replaced with a NASA-red dot + trailing ring that expands when hovering over interactive elements. The ring smoothly lerps behind for a polished, fluid feel.
2. ⏱️ Mission Countdown Timer — A new bar between the hero and the ticker shows a live countdown to Artemis III (Jan 1, 2027), counting down days, hours, minutes and seconds in real time with a gold-on-dark aesthetic.
3. 🚀 Space Quiz Modal — A fixed "Space Quiz" button (bottom-right) opens a fullscreen modal with 8 NASA trivia questions. It tracks your score, highlights correct/wrong answers in green/red, and loops when finished.
4. ☀/🌙 Light/Dark Theme Toggle — A sun/moon button (bottom-right cluster) toggles a light theme that inverts the dark palette to a soft navy-on-light-blue look, while keeping the brand colors intact.
5. ↑ Back-to-Top Button — A slim NASA-red butto
